# Identity Lifecycle Management Lab

## Objective

Design a Joiner, Mover, Leaver process for a fictional company.

## Scenario

Company: Contoso Manufacturing

Departments:
- HR
- Finance
- IT
- Operations

## Joiner Process

1. Create user account
2. Assign department
3. Assign security groups
4. Enable MFA
5. Assign license
6. Verify access

## Mover Process

1. Remove old permissions
2. Add new permissions
3. Validate least privilege

## Leaver Process

1. Disable account
2. Revoke sessions
3. Remove group memberships
4. Archive mailbox
5. Document termination

## IAM Concepts

- RBAC
- Least Privilege
- MFA
- Identity Governance
- Access Reviews
